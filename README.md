# bevy_rblx

Running Roblox games outside of the Roblox platform, archiving your own games and hosting them yourself 🎉

> [!WARNING]  
> **This project is currently in the pre-alpha development phase.** (very early!)
>
> You may notice incompatibilities, missing features or otherwise broken behavior. Feel free to ping me or create issues on this broken behaviour.

## Compiling
Clone the git repo, cd into it and you can get right into game development.
```
git clone https://github.com/radiantgurl/bevy_rblx.git
cd bevy_rblx
cargo build --release
```

## Installation

If you're on Linux, you can simply run `cargo install` after cloning the repository to install it instead.
